A wordpress plugin to make Survey/Questionaire. This plugin uses jQuery to move the load to the client's browser.

== Description ==
This plugin facilitates users to create Survey(s)/Questionaire(s) easily using drag-and-drop method. The clean simple design are crutial because there was no reason to increase their problem. There were instruction in (almost) every page, so the user can pick-up very fast.
In lite version, only plain text, Dropdown, and Textbox are present. In Pro version, there are extras features such as Import/Export survey questions, Radio-button Selection, Checkbox selection, and New page. New features and updates will be added in the future, IF there are a lot of users and consumer of this plugin.

== Installation ==

1. Upload `survey-maker-lite` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

= Does this plugin connects/interact to another website/server? =
No. We respect your privacy. We want to keep it that way.

= Does this means the data is in my database? =
Yes. We've created a new storage (table) inside your wordpress database. It won't harm your wordpress pages/posts/configurations.

= How can i remove question(s) from the survey =
Drag the question to the green box. Anything NOT on the question stack (on your left) will not be saved.

= When can i edit the question(s) from the survey =
Don't know. Sometime in the future i think. :)

= Does your survey plugin have Import/Export =
No. This lite version plugin does not have Import/Export function. But, the Pro version plugin does have it.

= Where i could get the Pro version plugin ? =
Nowhere for this time. I will update on you later.
